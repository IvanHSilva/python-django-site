🧩 Projeto Django Inicial

Este é um projeto básico desenvolvido com o framwork Django, utilizado como para aplicações web em Python.

⚙️ A estrutura segue o padrão do Django, contendo os arquivos principais de configuração, rotas e execução do servidor.

🗄️ Utiliza o banco de dados padrão SQLite, ideal para desenvolvimento inicial.

🚀 O projeto está preparado para ambiente local e pode ser facilmente expandido.

📦 Permite a criação de novos apps, além da definição de modelos, views e templates.

📈 Serve como uma base bem simples para estudos e para desenvolvimento de aplicações mais completas e escaláveis.
